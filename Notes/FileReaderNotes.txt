 File Reader 
 A java class used to read the character from the text file 
 Its reads character at one time from text file

 How Its Works 
         |
         |
         |
     1. File open 
     2.Read one character \
     3. Give ASCIII value 
     4. At the end it return -1=

  File → FileReader → Program

 __________


import java.io.FileReader;
import java.io.IOException;

public class FileReaderDemo {
    public static void main(String[] args) {

        try (FileReader fr = new FileReader("../data/filereaddemo.txt")) {

            int ch;
            while ((ch = fr.read()) != -1) {
                System.out.print((char)ch); TypeCasting important without this it will give ASCII Value
                
            }

        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
